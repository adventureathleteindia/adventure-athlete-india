## Admin Panel (`/admin`)

### Overview

Built-in admin dashboard at `/admin` route, protected by Supabase Auth. Single-user login (Atul only). Zero additional cost (Supabase free tier: 50,000 rows, 500MB).

**No CMS** â€” content management stays in Claude Code workflow (experience-entry.html â†’ Claude polish â†’ experiences.ts). Admin panel focuses on operations only.

### Security

| Layer | Implementation |
|-------|---------------|
| **Authentication** | Supabase Auth â€” email + password with bcrypt hashing |
| **Route Protection** | All `/admin/*` routes check for valid session server-side. No session = redirect to login |
| **Rate Limiting** | 5 failed login attempts â†’ 15-minute lockout (Supabase built-in) |
| **HTTPS** | Vercel serves everything over HTTPS by default |
| **API Protection** | Admin API routes require valid Supabase JWT token |
| **Single User** | No registration/signup page. Account created once during setup |
| **Session Expiry** | Auto logout after 24 hours of inactivity |
